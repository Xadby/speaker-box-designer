# Speaker Box Designer 🎵

Интерактивный конструктор акустических корпусов для динамиков с полной поддержкой 3D визуализации, расчетов и экспорта.

## 🚀 Возможности

### 🎨 3D Визуализация
- **Интерактивная 3D-модель** коробки с поддержкой:
  - Вращения мышью
  - Приближения/отдаления колесом
  - Сдвига правой кнопкой мыши
  - Сенсорного управления (мобильные устройства)

### 🎚️ Параметры в реальном времени
- **Размеры коробки**: от 200 до 1200 мм (ширина, высота, глубина)
- **Толщина материала**: от 10 до 30 мм
- **Динамик**: от 4 до 12 дюймов / 10-30 см
- **Порты**: полная настройка диаметра и длины
- **Щелевые порты**: независимые размеры

### 📦 Типы конфигураций
1. ✅ **Закрытый коробка** (Sealed)
2. 🔊 **Фазоинвертор** (Ported)
3. 📄 **Щелевой порт** (Slot Port)
4. 🎚️ **Бандпасс 4-го порядка** (Bandpass 4th Order)
5. 🎚️ **Бандпасс 6-го порядка** (Bandpass 6th Order)
6. 🔊 **Несколько динамиков** (Multi-Driver)

### 📐 Расчеты
- 📏 **Внутренний объем**:
  - В литрах
  - В см³
  - В дюймах³
- 🎯 **Резонансная частота** (Helmholtz resonator)
- 📊 **Автоматическое обновление** при изменении размеров

### 📤 Экспорт и Импорт
- 📄 **JSON**: Полное сохранение проекта
- 📋 **PDF**: Чертежи с размерами и 3D превью
- 📐 **DXF**: Для CAD программ
- 🔧 **STL**: Для 3D печати
- 📷 **PNG**: Скриншот 3D модели

### 💾 Сохранение
- Автосохранение в localStorage
- Импорт/экспорт проектов

## 🎮 Управление

### Мышь
- **Левая кнопка + движение** - Вращение модели
- **Правая кнопка + движение** - Сдвиг модели
- **Колесо мыши** - Приближение/отдаление

### Сенсорный экран
- **1 палец** - Вращение
- **2 пальца** - Масштабирование (pinch)
